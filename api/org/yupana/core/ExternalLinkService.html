  4.  abstract def transformCondition(condition: FlatAndCondition): Seq[ConditionTransformation]

    Transforms condition according to filter by this external link conditions.

    Transforms condition according to filter by this external link conditions. For example:

    linkX_fieldY = 'value1' AND
  linkY_fieldZ = 'value2' AND
  time >= TIMESTAMP '2019-02-01' AND
  time < TIMESTAMP '2019-02-28'

    might be transformed by ExternalLinkService for catalogX to:

    tagX in (`tagValue1`, `tagValue2`) AND
  linkY_fieldZ = 'value2' AND
  time >= TIMESTAMP '2019-02-01' AND
  time < TIMESTAMP '2019-02-28'

    So, the main idea is to provide more primitive conditions using values for supported fields (and maybe other fields as a context for catalog).

    condition

    condition to be transformed

    returns

    sequence of transformations applied to the initial condition, basically each transformation is a mapping from one expression to another. It should preserve time bounds even if there no conditions supported by this catalog.
